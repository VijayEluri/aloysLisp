Usage

For now the project is in pre-alpha stage, so not really usable and uncompleted. It's in creation process. But I put the code just for curiosity. Java src code is in /core/src. We use the Eclipse IDE with the last Java 6 version. The starting lisp script is /bin/lispsrc/class.lisp. This will disapear for first alpha version.
